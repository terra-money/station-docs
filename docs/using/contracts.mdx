# Contracts

Contracts are an advanced feature of Station, enabling developers to query, upload and instantiate CosmWasm smart contracts. 

## Enable developer mode

Before you can upload or instantiate a smart contract, you'll need to enable developer mode in Station. 

1. Visit [https://dashboard.station.money/](https://dashboard.station.money/) in your browser and connect your Station wallet extension. 

2. In the menu on the left, click **Settings**. 

3. In the advanced options, select **Enabled** for the developer mode setting. 

<img
    className="readme-img-lg"
    src="/img/screens/contracts/contracts-1.png"
    alt="contracts"
/>

4. Once enabled, you will see a "Contracts" option appear in the Station menu above the "Settings" option. 

5. Click **Contracts** to navigate to the contracts page.

    className="readme-img-lg"
